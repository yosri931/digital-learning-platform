
# 🎓 Digital Learning Platform (SaaS)

A full-stack **E-learning Platform** built with **Django + React**, designed to simulate a real-world SaaS product with authentication, course management, and lesson delivery.

---

## 🔥 Features

* 🔐 JWT Authentication (Login / Register)
* 👥 Role-Based Access Control (Student / Instructor)
* 📚 Course Management
* 🎥 Lesson System (Connected to Courses)
* 📈 User Enrollments (My Courses)
* ⚡ REST API (Django REST Framework)
* 🎨 Modern UI with React + Tailwind

---

## 🧱 Tech Stack

### Backend

* Django
* Django REST Framework
* JWT Authentication

### Frontend

* React (Vite)
* Axios
* Tailwind CSS
* Framer Motion

## ⚙️ Installation

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔗 API Endpoints

* `/api/token/` → Login
* `/api/courses/` → Get Courses
* `/api/enroll/` → Enroll User
* `/api/lessons/` → Lessons per Course
